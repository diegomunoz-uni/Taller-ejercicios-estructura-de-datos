# Taller-Estructuras-Lineales

## Nombre del estudiante
Diego Fernando Munoz

## Descripcion
Taller de estructuras de datos lineales en Java que cubre arreglos, ArrayLists y matrices.
Incluye 15 ejercicios organizados en paquetes siguiendo principios de POO y buenas practicas
de codigo limpio (DRY, KISS, YAGNI, SOLID).

## Estructura del proyecto
```
Taller-Estructuras-Lineales/
├── src/
│   ├── Main.java
│   ├── arreglos/       (Ejercicios 1-5)
│   ├── arraylist/      (Ejercicios 6-10)
│   └── matrices/       (Ejercicios 11-15)
└── README.md
```

## Instrucciones para ejecutar

### VS Code
1. Abrir la carpeta `Taller-Estructuras-Lineales` en VS Code.
2. Instalar la extension Extension Pack for Java.
3. Abrir `src/Main.java` y dar clic en Run.

## Requisitos
- Java 8 o superior
- Sin dependencias externas
